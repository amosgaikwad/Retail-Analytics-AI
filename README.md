# 📌 Project Overview

The Retail Analytics Platform is an interactive web application that enables users to explore retail sales data through modern dashboards, visualizations, and AI-powered forecasting.

The platform combines **Data Analytics**, **Machine Learning**, and **Business Intelligence** to transform raw retail data into actionable insights.
